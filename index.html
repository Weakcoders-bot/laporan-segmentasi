Pendahuluan

Segmentasi adalah proses memisahkan citra menjadi region bermakna. Thresholding memisahkan piksel berdasarkan intensitasnya — salah satu metode paling sederhana dan sering dipakai sebagai langkah awal.

Metode

  • Global thresholding: satu nilai ambang (T) untuk seluruh citra.
  • Otsu's method: otomatis mencari T yang memaksimalkan variansi antar-kelas.
  • Adaptive thresholding: ambang lokal yang dihitung dari neighborhood setiap piksel (Mean / Gaussian).

# Contoh sederhana: Otsu & Adaptive Thresholding
import cv2
import numpy as np
from matplotlib import pyplot as plt

image = cv2.imread('input.jpg', cv2.IMREAD_GRAYSCALE)
if image is None:
    raise SystemExit('Letakkan file input.jpg di folder yang sama.')

# 1) Otsu
_, otsu = cv2.threshold(image, 0, 255, cv2.THRESH_BINARY + cv2.THRESH_OTSU)
print('Threshold Otsu:', _)

# 2) Adaptive (Gaussian)
adaptive = cv2.adaptiveThreshold(image, 255, cv2.ADAPTIVE_THRESH_GAUSSIAN_C,
                                 cv2.THRESH_BINARY, 11, 2)

# 3) Tampilkan
plt.figure(figsize=(10,4))
plt.subplot(1,3,1); plt.title('Original'); plt.imshow(image, cmap='gray'); plt.axis('off')
plt.subplot(1,3,2); plt.title(f'Otsu (T={_})'); plt.imshow(otsu, cmap='gray'); plt.axis('off')
plt.subplot(1,3,3); plt.title('Adaptive Gaussian'); plt.imshow(adaptive, cmap='gray'); plt.axis('off')
plt.tight_layout(); plt.show()
Tips: untuk dokumen/foto dengan pencahayaan tidak merata gunakan Adaptive Thresholding. Untuk citra dengan histogram bimodal Otsu seringkali bekerja sangat baik.

Workflow praktis

  1. Preprocessing: konversi ke grayscale, smoothing (GaussianBlur) untuk mengurangi noise.
  2. Thresholding: pilih global/Otsu/adaptive sesuai karakter citra.
  3. Morphology: bersihkan hasil (opening/closing) untuk menghapus noise kecil atau mengisi lubang.
  4. Evaluasi: inspeksi visual dan/atau bandingkan dengan ground truth.
