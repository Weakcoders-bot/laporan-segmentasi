BAB I — Pendahuluan

Latar Belakang. Segmentasi citra merupakan tahap penting dalam pemrosesan citra digital yang bertujuan memisahkan objek-objek bermakna dari latar. Salah satu teknik segmentasi sederhana dan umum digunakan adalah thresholding, yang memanfaatkan perbedaan intensitas pada citra untuk memisahkan objek dan latar. Teknik ini banyak diterapkan pada berbagai domain seperti analisis citra medis, pengenalan karakter optik, dan inspeksi kualitas produk.

Rumusan Masalah. Bagaimana efektivitas metode Otsu dan Adaptive Thresholding dalam menghasilkan segmentasi yang akurat pada citra dengan kondisi pencahayaan seragam dan tidak seragam?

Tujuan Praktikum. (1) Mempelajari konsep thresholding; (2) Mengimplementasikan metode Otsu dan Adaptive Thresholding menggunakan Python-OpenCV; (3) Menganalisis perbedaan hasil pada variasi kondisi pencahayaan.

BAB II — Dasar Teori

Segmentasi Citra

Segmentasi citra adalah proses pembagian citra menjadi region-region yang memiliki kemiripan tertentu, misalnya intensitas atau tekstur. Tujuan utamanya adalah menyederhanakan dan/atau mengubah representasi citra menjadi sesuatu yang lebih bermakna dan lebih mudah dianalisis.

Thresholding

Thresholding adalah teknik segmentasi yang membandingkan intensitas piksel dengan sebuah nilai ambang (threshold). Jika intensitas lebih besar/lebih kecil dari ambang, piksel diklasifikasikan sebagai objek atau latar.

Otsu's Method

Metode Otsu adalah teknik memilih threshold global secara otomatis dengan memaksimalkan variansi antar-kelas (between-class variance). Metode ini bekerja baik ketika histogram citra bersifat bimodal.

Adaptive Thresholding

Adaptive Thresholding menghitung nilai threshold untuk setiap piksel berdasarkan statistik lokal (misal mean atau weighted sum menggunakan Gaussian) pada neighborhood-nya. Metode ini lebih tangguh menghadapi pencahayaan tidak merata.

BAB IV — Hasil dan Pembahasan

Bagian ini menampilkan contoh hasil segmentasi pada beberapa citra uji. Untuk setiap citra, tampilkan hasil Otsu dan Adaptive, lalu lakukan analisis perbedaan. Di sini disajikan pedoman penulisan hasil yang ideal:

  1. Deskripsikan karakter citra (kontras, adanya noise, pencahayaan)
  2. Sertakan nilai threshold Otsu yang ditemukan
  3. Bandingkan hasil Otsu dan Adaptive: where Otsu succeeds/fails

Contoh analisis singkat: Pada citra dengan histogram bimodal dan pencahayaan seragam, Otsu mampu memisahkan objek dengan baik. Namun pada citra dengan pencahayaan tidak merata, Adaptive Thresholding (Gaussian) menunjukkan hasil yang lebih konsisten dan sedikit lebih tahan terhadap bayangan.

BAB V — Kesimpulan dan Saran

Kesimpulan. Thresholding adalah metode sederhana namun efektif untuk segmentasi dasar. Pemilihan metode (Otsu vs Adaptive) harus disesuaikan dengan karakter citra; Otsu baik untuk kondisi seragam, Adaptive lebih cocok untuk kondisi pencahayaan variatif.

Saran. Untuk tugas lanjutan, kombinasikan thresholding dengan teknik lain (mis. edge detection, clustering, atau model berbasis pembelajaran) dan lakukan evaluasi kuantitatif menggunakan metrik (IoU, F1-score) terhadap ground truth.

Lampiran — Kode Program

Berikut adalah kode program yang digunakan. Simpan file ini sebagai segmentasi.py dan jalankan di lingkungan Python yang memiliki OpenCV.

# segmentasi.py
import cv2
import numpy as np
from matplotlib import pyplot as plt
from pathlib import Path

def run(input_path='input.jpg', out_dir='outputs'):
    p = Path(input_path)
    if not p.exists():
        raise SystemExit(f'File not found: {input_path}. Letakkan gambar uji bernama input.jpg di folder yang sama.')
    out = Path(out_dir)
    out.mkdir(parents=True, exist_ok=True)

    # Load & preprocessing
    img = cv2.imread(str(p), cv2.IMREAD_GRAYSCALE)
    blurred = cv2.GaussianBlur(img, (5,5), 0)

    # Otsu
    otsu_val, otsu = cv2.threshold(blurred, 0, 255, cv2.THRESH_BINARY + cv2.THRESH_OTSU)
    print('Otsu threshold value:', otsu_val)
    cv2.imwrite(str(out / 'output_otsu.png'), otsu)

    # Adaptive (Gaussian)
    adaptive = cv2.adaptiveThreshold(blurred, 255, cv2.ADAPTIVE_THRESH_GAUSSIAN_C,
                                     cv2.THRESH_BINARY, 11, 2)
    cv2.imwrite(str(out / 'output_adaptive.png'), adaptive)

    # Morphology cleanup (example)
    kernel = np.ones((3,3), np.uint8)
    cleaned = cv2.morphologyEx(adaptive, cv2.MORPH_CLOSE, kernel)
    cv2.imwrite(str(out / 'output_cleaned.png'), cleaned)

    print('Saved outputs in', out)
    plt.figure(figsize=(10,4))
    plt.subplot(1,4,1); plt.imshow(img, cmap='gray'); plt.title('Original'); plt.axis('off')
    plt.subplot(1,4,2); plt.imshow(otsu, cmap='gray'); plt.title('Otsu'); plt.axis('off')
    plt.subplot(1,4,3); plt.imshow(adaptive, cmap='gray'); plt.title('Adaptive'); plt.axis('off')
    plt.subplot(1,4,4); plt.imshow(cleaned, cmap='gray'); plt.title('Cleaned'); plt.axis('off')
    plt.tight_layout(); plt.show()

if __name__ == '__main__':
    import sys
    inp = sys.argv[1] if len(sys.argv) > 1 else 'input.jpg'
    run(inp)
